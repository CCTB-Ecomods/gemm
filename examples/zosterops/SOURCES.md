# DATA SOURCES

https://earthexplorer.usgs.gov/

## Forest cover

USGS LP-DAAC GFCC30TC
(https://lpdaac.usgs.gov/products/gfcc30tcv003/)

Landsat 5 & 7 (2015)

```
VALUE 	DESCRIPTION

0-100 	Percent of pixel area covered by tree cover
200 	Water
210 	Cloud
211 	Shadow
220 	Fill Value
```

## Elevation
